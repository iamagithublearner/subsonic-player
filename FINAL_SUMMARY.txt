QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════════

1. Install VLC (macOS example - also works on Linux):
   $ brew install vlc

2. Configure your Subsonic server:
   $ cp .env.example .env
   $ nano .env
   (Edit with your Subsonic URL, username, password)

3. Run the app:
   $ python3 main.py
   
   Then open: http://127.0.0.1:8000/docs

EXAMPLE API USAGE
═══════════════════════════════════════════════════════════════════════════════════

Add starred songs and play:
  POST /queue/add-starred
  POST /play

Skip to next song:
  POST /next

Set volume to 75%:
  POST /volume
  {"volume": 75}

Get current status:
  GET /status

Response example:
  {
    "is_playing": true,
    "current_song": {
      "title": "Song Name",
      "artist": "Artist Name",
      "album": "Album Name",
      "duration": 240
    },
    "current_time": 45.5,
    "volume": 75,
    "queue_size": 42,
    "current_index": 3
  }
