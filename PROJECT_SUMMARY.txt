PROJECT OVERVIEW
================
A Python application that streams music from a Subsonic server via HTTP API
with full playback control (play/pause, next/previous, volume, seek, queue).

ARCHITECTURE
============
├─ FastAPI Server (Port 8000)
├─ SubsonicClient (Subsonic API integration)
├─ VLCPlayer (Audio playback via VLC)
├─ Queue Manager (Song queue with threading)
└─ PlaybackManager (Orchestration layer)

KEY FEATURES
============
✓ Direct streaming from Subsonic server
✓ Complete playback control (play, pause, next, previous, seek, volume)
✓ Queue management (add, remove, clear, auto-advance)
✓ REST API with 20+ endpoints
✓ Thread-safe operations
✓ Automatic song advancement when finished
✓ Support for starred songs and playlists
✓ Interactive API documentation (Swagger UI)

QUICK START
===========
1. Install dependencies:
   pip3 install -r requirements.txt

2. Configure Subsonic credentials:
   cp .env.example .env
   # Edit .env with your Subsonic server details

3. Ensure VLC is installed:
   brew install vlc  (macOS)
   sudo apt-get install vlc  (Ubuntu)

4. Run the application:
   python3 main.py

5. Access API documentation:
   http://127.0.0.1:8000/docs

API ENDPOINTS (20+)
===================
Health & Status:
  GET  /health - Health check
  GET  /status - Playback status
  GET  /queue - Current queue

Playback Control:
  POST /play - Start playback
  POST /play/{index} - Play by queue index
  POST /pause - Pause
  POST /resume - Resume
  POST /next - Skip to next
  POST /previous - Go to previous
  POST /seek - Seek to position
  POST /volume - Set volume (0-100)

Queue Management:
  POST /queue/add - Add song
  POST /queue/add-playlist - Add entire playlist
  POST /queue/add-starred - Add starred songs
  POST /queue/remove/{index} - Remove song
  POST /queue/clear - Clear queue

Subsonic Data:
  GET /playlists - List playlists
  GET /starred - Get starred songs

TESTING INTEGRATION
===================
Use the included client.py for testing:

  python3 -c "
  from client import SubsonicPlayerClient
  
  client = SubsonicPlayerClient()
  client.add_starred()
  client.play()
  print(client.status())
  "

Or use cURL:
  curl http://127.0.0.1:8000/docs (Interactive docs)
  curl -X POST http://127.0.0.1:8000/queue/add-starred
  curl -X POST http://127.0.0.1:8000/play
  curl http://127.0.0.1:8000/status
